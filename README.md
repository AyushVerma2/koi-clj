# Koi-clj 

A testbed for invokable services in Ocean. Implemented in Clojure

## Usage

### Prerequisites:

- Install jdk1.8+
- Install [lein](https://leiningen.org)
 
Steps:

``` bash 
lein ring server
```

Navigate to localhost:3000
